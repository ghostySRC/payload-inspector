# Payload Inspector

> A sleek, high-performance Chrome Extension built for developers to easily capture, group, and inspect outgoing network payloads (POST, PUT, PATCH) right inside a clean, modern side panel.

## Features
- **Smart Origin Grouping:** Requests are bundled by domain (e.g., youtube.com) using an optimized accordion view to prevent UI lag.
- **Modern UI:** Built from scratch with a fully responsive, dark-themed interface, fluent animations, and smooth scrolling.
- **Heuristic Filtering:** Search and filter through domains and payload URLs instantly.
- **Multi-language Support:** Ready for localization with dynamic internationalization structures.

## Manual Installation (Developer Mode)
Since this project is 100% open-source, you can install it manually in less than a minute:

1. **Download** this repository as a ZIP file (click the green Code button above -> Download ZIP) and extract it on your computer.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Toggle the **Developer mode** switch in the top-right corner.
4. Click **Load unpacked** in the top-left corner and select the extracted folder containing the `manifest.json` file.
5. Open your Chrome Side Panel, select **Inspector**, and start debugging!
