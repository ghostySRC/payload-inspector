Release Notes
v1.0.2 (The Developer Power-Pack)
- Payload Diff Mode: Select two payloads to perform beautiful visual side-by-side JSON diffs.
- Network Type Filters: Instantly filter cards with [All], [Fetch/XHR], or [WebSockets] pills.
- Domain Blacklist: Stop background script from capturing muted domains to declutter panels.

v1.0.1 (The Mega Update)
- Copy as cURL / Fetch: Quick copy buttons inside expanded cards.
- Export All to JSON: Grouped domain export in toolbar.
- Deep Payload Search: Filters cards by key/value/body content.
- Interactive JSON Tree View: Expand/collapse nested JSON objects.
- Payload Size Indicator: Shows exact size (e.g. 1.4 KB) on card.
- JSON Syntax Highlighting: Professional dark/light themes.
- Pin/Favorite Payloads: Keeps cards pinned and protected.
- Response Status Matching: Colorized HTTP status badges.

v1.0.0 (Release)
- Initial stable release.
- Fully responsive UI for ultra-narrow side panels.
- Smart grouping and payload heuristics.
- Integrated secure donation bridge.
