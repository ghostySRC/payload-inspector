Release Notes
v1.0.3 (The Settings & Customization Update)
- All-in-One Settings Panel: Accessible via gear icon to manage themes, languages, notifications, capping, and blacklist.
- Compact View Mode: Densifies margins, fonts, and paddings to fit maximum payloads on screen.
- History Queue Cap: Smart garbage collector purges oldest unpinned requests at 50, 100, 200 thresholds.
- Audio Notification Chime: High-fidelity Web Audio API synth beep + gear pulsing animation on payload intercept.

v1.0.2 (The Developer Power-Pack)
- Payload Diff Mode: Select two payloads to perform beautiful visual side-by-side JSON diffs.
- Network Type Filters: Instantly filter cards with [All], [Fetch/XHR], or [WebSockets] pills.
- Domain Blacklist: Stop background worker from capturing blacklisted domains.

v1.0.1 (The Mega Update)
- Copy as cURL / Fetch: Quick copy buttons inside expanded cards.
- Export All to JSON: Grouped domain export in toolbar.
- Interactive JSON Tree View: Expand/collapse nested JSON objects.
- Payload Size Indicator: Shows exact payload size on cards.
- Pin/Favorite Payloads: Keeps cards pinned and protected.
